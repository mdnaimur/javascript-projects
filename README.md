# 🚀 JavaScript Learning Lab --  Projects

A collection of hands-on JavaScript projects built while learning modern JavaScript, browser APIs, and frontend engineering.

Each project focuses on a specific concept and is designed to improve problem-solving, coding practices, and understanding of how JavaScript works in the browser.

This repository focuses on learning:

* JavaScript (ES6+)
* DOM Manipulation
* Browser APIs
* Event Handling
* Asynchronous JavaScript
* Fetch API
* Local Storage
* Performance API
* Event Loop
* Memory Management
* Error Handling
* Functional Programming
* Modern Development Practices
